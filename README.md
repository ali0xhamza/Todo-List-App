# Todo-List-App

A Professional Todo List Application built with HTML, CSS, and JavaScript that allows you to manage tasks efficiently. It features task creation, editing, completion tracking, filtering, and persistent storage using localStorage.

# Features

* Add, edit, and delete tasks

* Mark tasks as completed or pending

* Filter tasks: All, Completed, or Pending

* Task statistics: Total, Completed, Pending

* Clear all tasks or only completed tasks

* Persistent storage using localStorage

* Responsive design for mobile and desktop

# Technologies Used

* HTML5

* CSS3

* JavaScript 

* localStorage for data persistence

# How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/ali0xhamza/Quiz-Application.git

2. Navigate to the project folder:

cd Todo-List-App

3. Open index.html in your browser to start using the app.

4. Add tasks: Type in the input field and click Add Task or press Enter.

5. Edit tasks: Click the Edit button on a task, modify, and press Enter or click outside to save.

6. Delete tasks: Click the Delete button on a task.

7. Filter tasks: Use the All, Pending, or Completed buttons.

8. Clear tasks: Use Clear Completed or Clear All buttons.
